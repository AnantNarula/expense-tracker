# Expense Tracker

A command-line expense tracker built in Python.

## Features
- Add, list and delete expenses
- Saves data between sessions using JSON
- Category breakdown and total summary

## How to run

python3 tracker.py add --amount 50 --category food --description "lunch"
python3 tracker.py list
python3 tracker.py summary
python3 tracker.py delete --id 1

## What I learned
- Python functions and file I/O
- JSON data storage
- Building a CLI with argparse
